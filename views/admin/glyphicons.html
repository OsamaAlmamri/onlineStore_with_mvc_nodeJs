چگونگی استفاده

به دلایل عملکرد، تمام آیکون ها نیاز به یک کلاس پایه و یک کلاس آیکون خاص دارند. که استفاده کنید، کد زیر را فقط در هر نقطه قرار دهید. اطمینان حاصل کنید که فاصله بین آیکون و متن برای پوشش مناسب را ایجاد کنید.

با اجزای دیگر مخلوط نکنید

کلاس های نماد نمی تواند به طور مستقیم با اجزای دیگر ترکیب شود. آنها نباید همراه با کلاس های دیگر در همان عنصر استفاده شوند. در عوض، <span> درونی دیگری اضافه کنید و کلاس نماد را به <span> بدهید.

فقط برای استفاده در عناصر خالی

کلاس آیکون ها فقط باید در عناصر حاوی محتوای متن و هیچ یک از عناصر دارای زیر مجموعه استفاده شود.

تغییر محل فونت آیکون

فوت آیکون های Bootstrap در محل ../fonts/ وجود دارد, که محل فونت ها در زمان کامپایل css ها می‌باشد. جابجایی و یا تغییر نام هریک از این فونت ها باید CSS را به یکی از سه شکل زیر تغییر دهید:

  • تغییر مقدار@icon-font-path و/یا @icon-font-name متغیر در فایل منبع LESS.
  • استفاده از گزینه URL نسبی که با استفاده از Less کامپایلر در دسترس است.
  • تغییر آدرس url() در فایل CSS کامپایل شده.

از بهترین گزینه که برای محیط کاری شما مناسب است استفاده نمایید.

آیکون‌ها دسترس پذیری

نسخه های مدرن از فناوری های کمک کننده محتوای CSS تولید شده و همچنین کاراکترهای Unicode خاص را اعلام خواهند کرد. برای جلوگیری از خروج ناخواسته و گیج کننده در خوانندگان روی صفحه (به خصوص وقتی که آیکون ها صرفا برای دکوراسیون استفاده می شوند)، ما آنها را با ویژگی aria-hidden = "true"پنهان میکنیم.

اگر از آیکون برای انتقال معنای (به جای فقط به عنوان یک عنصر تزئینی) استفاده کنید، اطمینان حاصل کنید که این معنا نیز به فن آوری های کمک رسانی منتقل می شود - مثلا شامل محتوای اضافی است که بصری با کلاس .sr-only پنهان شده است.

اگر شما کنترل هایی را با هیچ متن دیگر (مانند <button> که شامل یک آیکون است) ایجاد می کنید، همیشه باید مواد جایگزین را برای شناسایی هدف کنترل، به طوری که آن را برای کاربران فناوری های کمک کننده معنا داشته باشد. در این مورد، شما می توانید ویژگی های aria-labelرا بر روی کنترل خود اضافه کنید.

کپی 
<span class="glyphicon glyphicon-search" aria-hidden="true"></span>

مثال‌ها

از آنها در دکمه ها، گروه های دکمه برای نوار ابزار، ناوبری یا ورودی های فرم استفاده شده استفاده کنید.

Copy
<button type="button" class="btn btn-default" aria-label="Left Align">
  <span class="glyphicon glyphicon-align-left" aria-hidden="true"></span>
</button>

<button type="button" class="btn btn-default btn-lg">
  <span class="glyphicon glyphicon-star" aria-hidden="true"></span> ستاره
</button>

یک آیکون در هشدار استفاده می شود که نشان می دهد این یک پیام خطا است؛ با افزودن .sr-only متن برای انتقال این اشاره به کاربران فن آوری های کمک.

اخطار: یک ایمیل آدرس صحیح وارد نمایید.
کپی 
<div class="alert alert-danger" role="alert">
  <span class="glyphicon glyphicon-exclamation-sign" aria-hidden="true"></span>
  <span class="sr-only">خطا:</span>
  یک ایمیل آدرس صحیح وارد نمایید.
</div>
